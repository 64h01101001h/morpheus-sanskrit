// Visarga Sandhi Rule A1: (अ & visarga) standing for (अस्) changes to (ओ+avagraha) when followed by (अ). Here visarga changes to ओ and the अ of the second word changes to (avagraha)

(अ & visarga sf स्) + (अ) = (ओ) & (avagraha)
अजः	+ 	अपि	= 	अजोऽपि
Ajah	+ 	Api	= 	Ajopi <<== у них тоже проблема с аваграхой?

// Visarga Sandhi Rule C2: (अ & visarga) standing for अर् changes to अर् when followed by a (vowel or soft consonant except र्)

(अ & visarga sf र्) + (vow,sc-र्) = (अर्) & that (vow,sc-र्)
अजः	+ 	अपि	= 	अजरपि
Ajah	+ 	Api	= 	Ajarapi
